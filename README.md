# E-Commerce-Back-End
Building an e-commerce site by using Express.js server and Sequelize to interact with a MySQL database.

# Table of Contents

*Installation
*Usage
*Screenshot
*Video

# Installation

Run npm i

# Usage

Create database in MySQL run:
*mysql -u root -p
*source db/schema.sql

Seed File:
*npm run seed

Start express server
*node server. js
